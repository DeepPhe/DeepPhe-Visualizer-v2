# Display settings

Display controls change how the Cohort Explorer is arranged. They do not change active filters or patient results.

![Display settings](../assets/screenshots/end-user/display-settings.png)

## Layout

Use the layout button to switch between:

- cards arranged in individual columns; and
- a stacked flow layout.

## Density

- **Standard** provides the most spacing.
- **Compact** fits more filters on screen.
- **Compact+** exposes additional controls for card spacing and unused vertical space.

Use Standard when learning the interface and Compact when reviewing many filters.

## Theme

Choose a supplied theme from the **Theme** menu. **Theme Builder** allows advanced users to override individual theme colors.

## Font size

Use the minus and plus controls beside **Aa** to change the interface scale.

Display preferences are stored by the browser and can persist between visits.
