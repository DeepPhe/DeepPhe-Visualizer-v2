# Accessibility options

The Cohort Explorer includes controls for different visual and motion needs.

## High contrast

Select the contrast button to strengthen foreground and boundary contrast. Select it again to return to the current theme’s standard contrast.

## Reduced motion

Select the reduced-motion button to minimize interface animation and transitions.

The application also respects the operating system’s reduced-motion preference.

## Font size

Use the **Aa** controls to increase or decrease the interface scale.

## Keyboard use

- Use **Tab** and **Shift+Tab** to move between controls.
- Use **Enter** or **Space** to activate focused buttons and selectable values.
- Use **Escape** to close dialogs and restore or minimize the patient drawer.
- Focus indicators show the currently active control.

If a control is difficult to use with assistive technology, record the page, control label, browser, and assistive technology version when reporting the issue.
