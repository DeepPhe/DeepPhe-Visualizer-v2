# Export results

Use the download button in the Selected Patients table to create `cohort-patients.csv`.

## What is exported

The CSV contains:

- rows on the currently loaded patient page;
- the current search result within that page;
- the current sort order; and
- columns currently visible in the table.

Hidden columns are not exported.

## Export a useful file

Before selecting download:

1. navigate to the page you want;
2. clear or apply the patient-table search;
3. set the desired sort order;
4. open the column chooser and select the needed fields;
5. select **Export filtered cohort rows to CSV**.

## Exporting a large cohort

The patient drawer loads the cohort in pages. Export each required page separately. The current export does not automatically combine every cohort page into one file.

Treat exported files according to your organization’s policies for clinical and patient-identifying information.
