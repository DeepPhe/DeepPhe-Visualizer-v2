# Export results

Use the download button in the [Selected Patients table](patients-table.md) to create `cohort-patients.csv`.

## What is exported

The CSV contains, from the **currently loaded page** only:

- the rows on that page;
- the current **search** result within that page;
- the current **sort** order; and
- the columns that are currently **visible**.

Hidden columns are not exported, and the export does **not** automatically combine every page of the cohort.

## Export a useful file

1. Move to the page you want.
2. Clear or apply the table search.
3. Set the sort order you want.
4. Open the column chooser and show the fields you need.
5. Select **Export filtered cohort rows to CSV**.

## Exporting a large cohort

Because export covers one loaded page at a time, export each page you need separately.

:::caution

Exported files may contain patient-identifying information. Handle them according to your organization's policies for clinical and patient data.

:::
