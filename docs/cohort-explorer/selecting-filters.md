# Select and combine filters

## Select a value

Click a value’s label or bar. Selected values are highlighted, and the filter card header reports how many values are selected.

![Active filter selection](../assets/screenshots/end-user/active-filter-selection.png)

Click a selected value again to remove it.

## Select multiple values in one filter

Multiple values within one filter act as alternatives.

For example, selecting both `Stage II` and `Stage III` includes patients matching either stage.

## Combine different filters

Selections in different filters narrow the cohort together.

For example:

- Race: `White` or `Black`
- Stage: `Stage III`

The cohort must satisfy the race selection and the stage selection.

## Hierarchical values

Some clinical filters group detailed values under a parent. Use the expand control beside a parent value to reveal its children. Selecting a parent includes the values represented by that group.

Examples can include stage, grade, and behavior groupings.

## Age at diagnosis

Age may be grouped into ranges for easier selection. A selected range is translated into its represented ages when the cohort query is sent.

## Clear selections

- Remove one value by selecting it again.
- Remove all active criteria with **Reset filters**.

Resetting filters also clears the selected-patient result and returns the explorer to its unfiltered state.
