# Search and sort filter values

Select a filter card’s header to open the detail dialog. This is useful when a card contains many values or truncated labels.

## Search values

Enter text in **Search values**. The dialog limits the displayed values to labels containing the search text.

If no labels match, it displays **No matching values**.

The search changes only what is visible in the dialog. It does not change the cohort until you select a value.

## Sort values

Choose:

- **Count** to order values by patient frequency; or
- **Label** to order values alphabetically.

Use the adjacent direction button to switch between ascending and descending order.

Some filters use a clinical sequence by default—for example, stage values can follow their natural stage order.

## Make selections

Selections made in the dialog are the same selections shown in the compact card. Close the dialog when finished; the selections remain active.
